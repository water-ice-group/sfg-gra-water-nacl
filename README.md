# Supplementary data for the paper "Breaking the Air-Water Paradigm: Ion Behavior at Hydrophobic Solid-Water Interfaces"

This repository contains supplementary data supporting the findings of the paper:

"Breaking the Air-Water Paradigm: Ion Behavior at Hydrophobic Solid-Water Interfaces" (Xavier R. Advincula, Kara D. Fong, Yongkang Wang, Christoph Schran, Mischa Bonn, Angelos Michaelides, Yair Litman)

## Contents
- `input-simulations`: Example input files for the MD simulations performed in this work. This folder includes:
  - `equi-ff-example`: Scripts used to generate reasonable initial structures with a classical force field.
  - `piston-example`: Scripts employing the piston approach to equilibrate the pore width such that the system reaches 1 atm.
  - `production-example`: Scripts for equilibration on the NNP potential energy surface, followed by the production runs reported in the manuscript.
- `nnp-data`: The neural network potential (NNP) model employed in this study.
